O elemento table (tabela)

O elemento table (tabela) representa dados tubulares, ou seja, informações que são representadas em linhas e colunas

exemplos

Tabela simples com cabeçalho

Primeiro nome Sobrenome
Eduarda Neves

Elementos de tabela

  • table - tabela;
  • th (table header) - cabeçalho da coluna;
  • tr (table row) - linha da tabela;
  • td (table data) - valor/conteúdo da célula;
