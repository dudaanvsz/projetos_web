Listas não ordenadas

A tag ul (unordered list - lista não ordenada) representa uma lista de itens sem ordem rígida, por exemplo ingredientes em uma receita.

A tag li (list itens - itens de lista) representa um item que faz parte de uma lista.

Exemplo de lista

  • peimeiro item
  • segundo item
  • terceiro item
    • subitem

Listas Ordenadas

A tag ol (ordered list - lista ordenada) representa uma lista de itens numerados sequencialmente.

  1. primeiro item
  2. segundo item

Atributos

reversed (ordem reversa)
Exibe os itens da lista ordenada em ordem revrsa

  1. item mais importante
  2. item menos importante

start (início)
especifica o valor inicial para a numeração da lista.

  1. Item 5
  2. Item 6

    type (tipo)
    Indica o tipo de numeração:
    a - Indica letras minúsculas
    A - Indica letras maiúsculas
    i - indica algarismos romanos minúsculos
    I - indica algarismos romanos maiúsculos

    1. peimeiro item
    2. segundo item
